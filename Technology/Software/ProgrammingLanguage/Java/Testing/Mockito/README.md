## [Mockito](https://site.mockito.org/)

### Setup

#### Gradle

```
repositories { jcenter() }
dependencies { testCompile "org.mockito:mockito-core:2.+" }
```

#### Maven

```
<dependency>
  <groupId>org.mockito</groupId>
  <artifactId>mockito-core</artifactId>
  <version>3.1.0</version>
</dependency>
```
