## [JUnit5](https://junit.org/junit5/)

### Setup

#### Gradle

```
testImplementation 'org.junit.jupiter:junit-jupiter:5.5.1'
```

#### Maven

```
<dependencies>
  <dependency>
    <groupId>org.junit.jupiter</groupId>
    <artifactId>junit-jupiter</artifactId>
    <version>5.5.1</version>
    <scope>test</scope>
  </dependency>
</dependencies>
```
