## [SLF4J](http://www.slf4j.org/)

### Setup

#### Gradle

```
repositories { jcenter() }
dependencies { implementation "org.slf4j:slf4j-log4j12:1.7.28" }
```

#### Maven

```
<dependency>
  <groupId>org.slf4j</groupId>
  <artifactId>slf4j-log4j12</artifactId>
  <version>1.7.28</version>
</dependency>
```
