## [Installing Helm](https://helm.sh/docs/intro/install/)

### Linux/Windows

```
$: curl https://raw.githubusercontent.com/helm/helm/master/scripts/get-helm-3 > get_helm.sh
$: chmod 700 get_helm.sh
$: ./get_helm.sh
```

```

```
