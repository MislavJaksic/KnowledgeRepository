#### edit

Edits a resource using a `KUBE_EDITOR` or `EDITOR`.  

```
$: kubectl edit Resource-Type/Resource-Name
```

#### rollout

Manages rollout of:
* `Deployment`
* `DaemonSet`
* `StatefulSet`

```
$: kubectl rollout restart Specific-Type/Resource-Name
```
