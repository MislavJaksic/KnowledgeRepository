## [Deployment](https://www.jaegertracing.io/docs/1.16/deployment/)

`Docker` images on `Docker Hub`:
* `jaeger-agent`
* `jaeger-collector`
* `jaeger-query`
* `jaeger-ingester`

Orchestration templates exist for:
* `Kubernetes`
* `OpenShift`

### Configuration Options

Order of priority for Jaeger binaries:
* command line arguments
