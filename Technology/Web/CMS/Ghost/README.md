Features:
* simplicity
* monetization
* headless Node.js CMS
* integrations

### Setup

[In Cloud or locally](Docs/Setup)

### Core Concepts

Posts:
* Primary entity-type
* Created in the editor
* Can have `Card`s
* Grouped

Pages:
* Pages are special posts
* Static and independent pages like `About` or `Contact`
* They are not crosslinked
