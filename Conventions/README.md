## GitHub convention

### Markdown conventions

The top title is:  
```
##
```

Command prefix:  
```
$:
```

Variable name:  
```
Separated-By-Minus
OR
_underscored
```

Line comment:
```
#
```

Output comment:  
```
#->
```

### Text conventions

```
DO

DON'T
Break up sentences if they are longer then X characters.  
```
