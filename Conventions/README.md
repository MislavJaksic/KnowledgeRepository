## GitHub convention

### Folder conventions

Folder names have no special characters and are CamelCased.  

### Text conventions

The top title is:  
```
##
```

Command prefix:  
```
$:
```

Variable name:  
```
Separated-By-Minus
OR
_underscored
```

Line comment:
```
#
```

Output comment:  
```
$: _command  #->
```

Long output comment:
```
$: _command  #->
  # -a _descriptionA
  # -b _descriptionB
```

```
DO
Horizontally align text if it fits the screen.  

DON'T
Break up sentences if they are longer then X characters.  
```
